Made by @Rekt05 on Discord.

This project scrapes all of the Hitman speedrun.com boards weekly on Sunday at Midnight UTC+2 and stores the data into JSON files that you can find on GitHub.

This site reads the JSON files and displays the platform distribution based on a few options.
I made this because there has been some discussion about downpatching and its effects on console players, but I thought it could be interesting to some.
